Peso e estilo das fontes

Peso quando a fonte é mais magrinha ou mais gordinha. Não é possível usar fontes simples. Ex Work Sans.
Fonte de exemplo: WORK SANS - font-weight: normal; /*Declaração do peso da fonte. lighter (mais leve), normal (normal), bold (Mais grossa) , bolder (Mais negrito) Possível colocar variações, ex 100 a 900*/

Configurações mais usadas:
font-family: 'Arial', Helvetica, sans-serif;
font-weight: bolder;
font-size: 3em;
font-style: italic;
/*Configurações usadas com frequência. Esta fonte foi meu exemplo, mas do curos foi a WORK SANS*/

Shorthand (MãoZinha)

Shorthand font:
sequência:
font-style -> font-weight -> font-size -> font-family
font: italic bolder 3em 'work Sans', sans-serif;

TAG font-weight para negrito, font-style para itálico e text-decoration: underline para sublinhado.
