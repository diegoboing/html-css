Links externos

Para criar links externos, será necessário selecionar a parte do texto da sua preferência e criar o código como por ex:
Acesse meu canal no YoutYube clicando em: RDKeletronica

A criação do código é: Envelopar a tag A (significado Ancora) e terá href="link do site/Aqui CRTL+Espaço para selecionar o caminho" com rel="next" (ir para a próxima página se utilizar link INTERNO, no caso da mesma página) rel="external" (página externa) target="_BLANK" (abre uma nova guia)

Este é o código que leva para links externos

Usando links internos

Link interno liga uma pasta que está dentro do nosso servidor, no caso onde a página está hospedada. Link interno trata de link da nossa própria página.

Para executar esse código basta selecionar o texto que será o hyperlink EX: VOLTAR A PÁGINA INICIAL e envelopar a tag A (significado Ancora) e terá href="link do site/Aqui CRTL+Espaço para selecionar o caminho" com rel="next" (ir para a próxima página se utilizar link INTERNO, no caso da mesma página)

Tem também a tag NOFOLLOW

A tag NOFOLLOW serve pra quando você não dá o aval para um site, ex: uma empresa pagou para ter a marca no site, mas isso não significa que você confia ou que você aprova o que eles fazem, PARA O MECANISMO DE BUSCA não procurar lá, etc...
