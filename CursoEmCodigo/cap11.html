Imagens dinâmicas e que se adaptam sozinhas

Necessário utilização do programa GIMP

  • GIMP

Aviso importante: para edição das imagens é possível utilizar outros programas, porém este programa GIMP é gratuito.

Procedimento para edição da imagem

Com programa aberto, criar 3 arquivos com largura e altura de 300px, 700px e 1000px:

  • Criar uma nova imagem;
  • Largura 300px, 700px e 1000px
  • Resolução de 80px
  • exportar a imagem para a mesma pasta dos arquivos HTML

Para fins de prática, seguir o procedimento abaixo:

  • Primeiro devemos criar o arquivo nas resoluções informadas acima e colocar um P (pequeno), M (médio) e G (grande) para ser compreendido no momento da demonstração.
  • Todos os arquivos deverão ser salvos na pasta do HTML
