Estilo externo via link

Os estilos externos, são estilos criados em CSS e que são chamados como links. Este estilo é o mais usado quando trata-se de códigos que possuem várias páginas.
Vantagens:

  • Fácil manutenção.
  • Ajustes de forma imediata em todas as páginas.
  • Possível criação de vários estilos.
  • Para adicionar ao código deve ser chamado via link em forma de seletor.

Demonstração do código

O estilo deve ser criado no formato .CSS.
Abaixo um código criado no estilo externo .CSS. 

 
        
            @charset "UTF-8";
            *{
                font-family: Arial, Helvetica, sans-serif;
                /*height: 100%;*/
            }
            body {
                background-image: linear-gradient(to right,  #31AFE0, #E47632);
            }
            main {
                background-color: white;/*Fundo da página MAIN*/
                border-radius: 10px; /*Arredonda o conteúdo da borda, ao invés de deixar quadrado*/
                box-shadow: 5px 5px 15px rgba(36, 36, 36, 0.842); /* parâmetros, 1 quanto anda para o lado, 2 quanto vai para baixo, 3 quanto vai espalhar e o 4 quanto de transparência terá e a cor*/
                width: 780px; /*largura do quadrado*/
                padding: 10px;
                margin: auto;/*centraliza o conteúdo*/
            }
            h1 {
                color: #2F47AD;
                text-align: center;
            }
            h2 {
                color: #31AFE0;
                text-shadow: 1px 1px 1px #144a5fbd;
                }
            h3 {
                color: #31AFE0;
                text-shadow: 1px 1px 1px rgba(87, 87, 87, 0.5);
                }
            p {
                text-align: justify;
            }

Código na cabeça / seletor

Formato do link para chamar o .CSS.
Exemplo abaixo:

link rel="stylesheet" href="site2.css"

Capítulo 1.2

É possível mesclar todas os estilos na mesma página, mas cada um com sua peculiaridade.

Capítulo 2

Capítulo 2.1

Estilo externo é um dos mais indicados caso o código da página seja grande, já o estilo interno ou local tem desvantagens e é o tamanho do código caso utilize estilo interno/local.
