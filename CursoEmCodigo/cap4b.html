Configurações

Após instalado o VGS, deve abrir e então em CONFIGURAÇÕES e alterar:

  • Idioma para português
  • Word Wrap - Quebra de linha para o programa de forma automática, para ON.
  • Instalar o pacote JavaScript.
  • Instalar o pacote CSS3
