Git e GitHub

Git - Programa para atualização das páginas do código. Ex: rev 1, rev2, etc. Mantém os backups no PC. Chamado de repositório local.

GitHub - Server para backup do Git. Mantém os backups na nuvem. Repositório remoto.

Repositório local/remoto

  • Instalar o Github no PC e escolher a pasta que será utilizada para armazenar os arquivos.
    • Na versão 3.1 quando adicionado os arquivos na pasta, é necessário dar um COMMIT e colocar um novo nome para o arquivo para posterior envio.
