Demonstração das imagens dinâmicas

Apresentação das imagens dinâmicas

O código utilizado foi:

TAG PICTURE, e função SOURCE MEDIA para alteração do tamanho das imagens.

  • Primeiro chama as tags acima citadas.
  • Em IMG SRC será necessário adicionar a maior imagem.
  • Agora usa a função SOURCE MEDIA e altera para max-width para utilizar o parâmetro DO MAIOR PARA O MENOR. Para o exemplo utilizamos na primeira linha para a image P 730px e para a outra linha imagem M o tamanho 1030.
  • Lembrando que as imagens foram criadas nos tamanho 300, 700 e 1000.
  • Basta abrir o navegador e fazer os testes, aumentando e diminuindo seu tamanho para a alteração automática das imagens.
