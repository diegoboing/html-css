Importando Fontes de outros locais

Acessar o site DAFONT, deve escolher a fonte, fazer o download e adicionar numa pasta junto ao código .HTML.
Depois, executar as linhas de comando acima para importação da fonte.

Adicionando fontes de outra pasta

Um exemplo do código que deve estar na cabeça: src: url('fonts/EasterFont St.ttf') format('opentype');
Fonts é o nome da pasta neste código.

Adicionando mais de uma fonte no seletor

Para adicionar mais de uma fonte, basta colocar mais seletores @font-face, adicionar o nome a o source...

Exemplo dos códigos para utilizar

Seletor para configurar a fonte: 
@font-face {
            font-family: 'EasterFont St';
            src: url('fonts/EasterFont\ St.ttf') format('opentype');
            
            Para adicionar uma segunda fonte, necessário um novo seletor:
            @font-face {
                font-family: 'use';
                src: url('fonts/SILVERS\ PERSONAL\ USE.ttf') format('Truetype');
    
Código para escolher a fonte no seletor:
       
                
 body {
                    font-family: Arial, Helvetica, sans-serif;
                    font-size: 16px;
                    color: black;
                }
                h1 {
                    font-family: 'EasterFont St';
                    font-size: 3em;
                    font-weight: normal;
                    /*font-weight: normal - Tirar o negrito das letras ou colocar...*/
                }
                h2 {
                    font-family: 'use';
                    font-size: 2em;
                    font-weight: normal;
                 }
