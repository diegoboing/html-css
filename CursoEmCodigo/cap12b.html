Estilos locais / internos

São estilos CSS que são criados na cabeça do código. Este estilo é muito utilizado em códigos que possuem poucas páginas. É possível configurar fonte, cor da fonte, fundo, alinhamento, etc... tudo na cabeça do código.
Este estilo é baseado em seletores.
Exemplo abaixo:

 style> 
        body {
            background-color: lightsteelblue;
            font-family: Arial, Helvetica, sans-serif;
            font-size: 15px;
        }
        h1 {
            color: darkblue;
        }
        h2 {
            color: darkred;
            background-color: lightblue;
        }
        p {
            text-align: justify;
        }
    
    
Capítulo 1.1

    
Este estilo é baseado em seletores, é utilizado em uma ou duas páginas no máximo, pois caso a necessidade de troca de cor, fundo, etc, torna-se mais difícil e demorada, pois precisa fazer um a um.
