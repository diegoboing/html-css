Três tipos de alinhamentos

A direita, a esquerda e center.
a TAG CENTER já está obsoleto e deve ser feita por CSS.

Os alinhamentos devem sempre ser utilizados por folhas de estilos (CSS) e não por HTML.

        style>
        body {
            font: normal 16px arial, verdana, serif;/*declaração para configuração da fonte.*/
        }
        h1 {
            text-align: center;/*Alinhamento ao centro*/
        }
        h2 {
            text-align: right;/*Alinhamento a direita*/
        }
        p {
            text-align: justify; /*Justificado*/
            text-indent: 30px;/*Parágrafo*/
        }
    /style>
