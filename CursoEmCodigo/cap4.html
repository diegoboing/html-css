Os softwares que serão utilizados são:

  • Visual Studio Code (VGS)

  • GIMP

  • HANDBRAKE

  • WinRAR - Software para descompactar os arquivos

Procedimento para a instalação de todos os programas

  1. Clicar no link para download dos programas.
  2. Descompactar o arquivo, clicando com botão direito sobre o arquivo, Descompactar.
  3. Acessar a pasta do arquivo descompactado.
  4. Executar o arquivo clicando duas vezes.
  5. Clicar em avançar até concluir a instalação.

Na lista de programas para download, está disponível o programa WinRAR para descompactar os arquivos que serão baixados.

Ou você pode clicar no link: WinRAR
