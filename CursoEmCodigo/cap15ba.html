Isso se chama Pseudo-Classes utilizando o HOVER

Fim do exemplo.

Código abaixo no CSS sendo configurado via link.CSS

        style>
        body {
            font-family: Arial, Helvetica, sans-serif;
        }

        div > p { /*div tem um filho que é um parágrafo. Irá esconder o parágrafo quem tem nas DIV.*/
            display: none;
        }

        div:hover > p { /*Representação de filho. Mostra o texto oculto quando passa o mouse por cima do texto da div. Nesse exemplo seria o PASSE O MOUSE AQUI na div; */
            display: block;
            color: white;
            background-color: red;
            width: 300px;
        }

        div:hover {
            color: blue;

        }
    


    
Exemplo do código em HTML

    
O código abaixo é feito no parágrafo no HTML,no h1, h2 ou p. Usando a tag SPAN envelopado é possível colocar a mesma configuração do CSS criada. 
No exemplo abaixo está sendo utilizado class="basico", sendo selecionado somente a palavra ou frase desejada. O nome basico, avancado, etc é criado para fim de identificação.

    
        
            h1 id="principal">Passos para criar alinhamentos para diferentes TAGs /h1

               h1>Aprendendo HTML /h1
               h2 class="basico">HTML básico
